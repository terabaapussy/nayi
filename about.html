About Our Store

Welcome to our Amazon Clone, a demonstration e-commerce platform built with modern web technologies.

Our Mission

To provide a seamless shopping experience with a focus on simplicity and user-friendly design.

Features

  • Product catalog with dynamic loading
  • Shopping cart with localStorage persistence
  • Checkout process with Web3Forms integration
  • Admin dashboard for product management
  • Responsive design for all devices

Technology Stack

This project was built using:

  • HTML5 and semantic markup
  • Tailwind CSS for styling
  • Vanilla JavaScript for functionality
  • Web3Forms for contact form handling
